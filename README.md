# 🌱 Sustainability ROI Calculator

A web-based **Sustainability ROI Calculator** developed using **HTML, CSS, JavaScript, and Bootstrap 5**. This application helps organizations evaluate the financial, environmental, and reputation impacts of sustainability initiatives through an interactive and user-friendly dashboard.

---

## 📖 Project Overview

The Sustainability ROI Calculator is designed to support organizations in assessing the return on investment (ROI) of sustainability initiatives by combining financial analysis with ESG (Environmental, Social, and Governance) metrics.

The application allows users to input company information, evaluate sustainability initiatives, analyze reputation benefits, and visualize key performance indicators.

## 📊 Modules

### 1. Home
- Landing page
- KPI cards
- ESG overview
- Sustainability goals
- Platform features

### 2. Company Inputs
Collects company information including:

- Annual Revenue
- Employees
- Energy Consumption
- Water Consumption
- Waste Generation
- Scope 1 Emissions
- Scope 2 Emissions
- Scope 3 Emissions
- WACC
- Tax Rate
- Inflation Rate

---

### 3. Initiative Planning

Manage sustainability initiatives such as:

- ESG Policy Framework
- Supplier ESG Assessment
- Water Recycling
- Solar Installation

- Green Procurement
- Zero Waste Program
- Women Leadership Program

Each initiative records:

- Investment Cost
- Operating Cost
- Revenue Impact
- Risk Avoided
- Reputation Benefit
- Carbon Reduction
